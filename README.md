## 🎯 Overview

Experience the taste of KFC Uzbekistan through this comprehensive frontend website that delivers a **professional restaurant experience** with fully responsive design and intuitive navigation. Built as a showcase project demonstrating modern web development techniques and clean code practices.

### 📂 **What's Inside This Website?**
- **🏠 Multi-Page Layout** - Complete restaurant website structure
- **📱 Fully Responsive** - Perfect on all devices and screen sizes  
- **🎨 Modern UI/UX** - Clean, professional design with smooth interactions
- **🍔 Restaurant Features** - Menu, pricing, staff, and contact information
- **⚡ Fast Loading** - Optimized performance with vanilla JavaScript

## 🛠️ Installation & Setup

### **1. Clone the Repository**
```bash
git clone https://github.com/GhostKX/KFC-Frontend-Website-Project.git
cd KFC-Frontend-Website-Project
```

### **2. Local Development**
```bash
# Option 1: Simple file opening
open index.html  # macOS
start index.html  # Windows
xdg-open index.html  # Linux

# Option 2: Local server (recommended)
python -m http.server 8000  # Python
npx serve .  # Node.js
live-server  # Live Server extension
```

### **3. Development Tools**
```bash
# Recommended VS Code Extensions
- Live Server
- HTML CSS Support
- JavaScript (ES6) code snippets
```
